欢迎使用 **灵珠 Agent Bridge**。这是一个用于将灵珠协议与第三方 Agent 协议（OpenAI Compatible 或自定义 HTTP）进行映射与转换的中间件服务，并提供可视化绑定与沙箱调试能力。

## 你能做什么

- 用可视化方式管理「灵珠 → 目标协议」绑定
- 通过模板映射任意 HTTP/JSON 接口
- 调试完整的请求/响应流（含 SSE 流式）
- 通过 Bridge API 将灵珠请求转发到目标模型/服务

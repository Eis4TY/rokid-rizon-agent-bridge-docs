## 环境要求

- Node.js >= 18
- npm 或 yarn

## 安装依赖（清华源）

```bash
npm config set registry https://registry.npmmirror.com
npm install
```

## 启动开发服务器

```bash
npm run dev
```

打开 `http://localhost:3000`。

## 最小可用流程

1. 进入首页，点击「新建绑定」。
2. 选择 `OpenAI (HTTP)` 或 `Custom (HTTP)`。
3. 填写 `Target URL` 与 `Auth Key`（如需）。
4. 保存后进入沙箱进行验证。
5. 调用 Bridge API 进行端到端联通。

## Bridge API 示例

```http
POST /api/bridge/{agentId}
Authorization: Bearer <settings.json 中的 apiKey>
Content-Type: application/json

{
  "message_id": "msg_123",
  "agent_id": "agent_123",
  "message": [
    { "role": "user", "type": "question", "text": "你好" }
  ]
}
```

> `agentId` 为绑定记录的 ID。
